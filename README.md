# 🎙️ Python Voice Assistant

A simple AI-powered Voice Assistant built using Python that listens to voice commands, responds with speech, and performs basic tasks like opening websites, telling the time, and answering questions using the Gemini AI API.

This project was developed to practice Python programming, speech recognition, text-to-speech, and AI integration.

---

## ✨ Features

- 🎤 Voice command recognition
- 🔊 Text-to-Speech responses
- 🌐 Open websites like YouTube and Google
- 🕒 Tell the current time
- 🤖 AI-powered responses using Google Gemini
- ❌ Exit the assistant using voice commands

---

## 🛠️ Technologies Used

- Python 3.x
- SpeechRecognition
- PyAudio
- pyttsx3
- Google Gemini API
- Webbrowser module
- Datetime module

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/voice-assistant.git
```

### 2. Navigate to the project folder

```bash
cd voice-assistant
```

### 3. Create a virtual environment (Optional)

**Windows**

```bash
python -m venv venv
venv\Scripts\activate
```

### 4. Install dependencies

```bash
pip install -r requirements.txt
```

---

## 🔑 Gemini API Setup

Create a `.env` file (or update your configuration file) and add your Gemini API key.

Example:

```text
GEMINI_API_KEY=YOUR_API_KEY
```

Replace `YOUR_API_KEY` with your actual API key.

## ▶️ Run the Project

```bash
python assistant.py
```
